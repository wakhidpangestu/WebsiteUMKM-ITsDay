# 🥟 Siomay Alek – Landing Page UMKM

Selamat datang di repository resmi untuk **Siomay Alek**, UMKM kuliner khas Indonesia yang menyajikan siomay lezat dengan cita rasa autentik dan tampilan modern. Website ini dirancang sebagai media promosi digital untuk memperkuat branding, memperluas jangkauan pelanggan, dan meningkatkan penjualan secara online.

---

## 🎯 Tujuan Website

Website ini bertujuan untuk:

- Meningkatkan visibilitas online Siomay Alek
- Memberikan informasi lengkap tentang produk dan promosi
- Mendorong pelanggan untuk melakukan pemesanan langsung
- Menampilkan brand Siomay Alek secara profesional & menarik

---

## ✨ Fitur Unggulan

- 🌟 **Desain Modern** – Glassmorphism, warna hijau segar, dan tata letak responsif
- 🌓 **Dark & Light Mode** – User dapat memilih tampilan siang/malam
- 🖼️ **Visual Produk** – Foto siomay berkualitas tinggi yang menggoda
- 🎁 **Promo Menarik** – Gratis Ongkir & Diskon hingga 50%
- 📱 **Order Sekarang** – Call-to-action jelas dan link pemesanan langsung
- 🔗 **Media Sosial & Kontak** – Tautan ke platform digital bisnis

---

## 🛠️ Teknologi yang Digunakan

| Teknologi | Deskripsi |
| --- | --- |
| **Next.js** | Framework React untuk SSR & SPA |
| **Tailwind CSS** | Utility-first CSS untuk styling cepat |
| **HTML5/CSS3** | Struktur dan gaya dasar |
| **JavaScript (ES6+)** | Interaktivitas & logika front-end |
